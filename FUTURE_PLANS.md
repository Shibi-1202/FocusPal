# Future Plans

These features were intentionally removed from the current release scope to keep the app smaller and more focused:

- Analytics dashboard
  Current release keeps task planning and execution only. Daily summaries, completion charts, streak dashboards, and priority breakdown views are deferred.

- AI insights
  Productivity insights, burnout risk analysis, overcommitment warnings, and related AI settings are deferred.

- Real AI model integration
  Replace the current hardcoded/local heuristic AI engine approach with a real model-backed implementation for task parsing, insights, recommendations, and related intelligence features.

- Audio customization
  Custom task sounds, break sounds, volume controls, and audio asset selection are deferred.

Potential future re-entry criteria:

- The release task flow and widget interaction are stable end to end.
- Core planning, task execution, Pomodoro, lookup, and breaks are verified in the desktop app.
- Removed features are rebuilt as isolated modules instead of being mixed into the main renderer flows.
